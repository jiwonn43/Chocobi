**학사일정 AI 챗봇 & D-Day 서비스**

## 주요 기능
**AI 학사일정 챗봇**:로컬 AI(gemma4)를 탑재하여 "이번 주 일정 있어?", "내일 시험이야?" 같은 질문에 간결하게 답변
**최근 학사일정 D-Day**:오늘 날짜와 가장 가까운 학사 일정을 D-Day 카드 형태로 제공

## 기술 스택

* **Language:** Python 3.10+
* **Framework:** Streamlit
* **AI Engine:** Ollama (`gemma4`)
