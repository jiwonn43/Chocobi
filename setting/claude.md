# CLAUDE.md

## Project

* 이 프로젝트는 Python을 사용한다.
* node.js 언어가 아닌 Python으로 UI, 서비스 등의 모든 코드를 작성한다.

## Code Style

* 변수명과 함수명은 의미가 명확하도록 작성한다.
* 하나의 함수는 하나의 책임만 갖도록 구현한다.
* 중복 코드를 만들기보다 기존 코드를 재사용한다.
* 불필요하게 복잡한 코드를 작성하지 않는다.

## Comments

* 모든 코드 주석은 한국어로 작성한다.
* 불필요한 주석은 작성하지 않는다.

## Dependencies

* 새로운 라이브러리는 반드시 필요한 경우에만 추가한다.
* 기존 라이브러리로 해결 가능한 경우 새로운 의존성을 추가하지 않는다.

## Code Changes

* 기존 코드를 최대한 재사용한다.
* 기존 기능을 임의로 삭제하거나 변경하지 않는다.

## Project Structure

* 기존 프로젝트 구조를 유지한다.
* 새로운 파일은 반드시 필요한 경우에만 생성한다.

## Testing

* 새로운 기능을 추가하면 가능한 경우 pytest 기반 테스트를 함께 작성한다.
* 기존 테스트를 불필요하게 수정하지 않는다.
